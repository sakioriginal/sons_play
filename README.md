# A-Frame VR Throw Monster Demo

Meta Quest 3S のブラウザで試すための、A-Frame製VRボール投げデモです。

## 使い方

1. このフォルダを GitHub Pages にアップロード
2. Quest 3S のブラウザでURLを開く
3. 右下のVRボタンからVRモードへ
4. コントローラーのトリガー、またはグリップでボールを掴む
5. 手を振って離すと投げる
6. 小モンスター風ターゲットに当たるとスコア加算

## PCでの確認

PCでは画面クリックでボールを発射できます。
WASDで移動、マウスで視点移動できます。

## 注意

- ポケモンそのものは使っていません。オリジナルのターゲットです。
- QuestブラウザのWebXR対応状態により、挙動が変わる場合があります。
- HTTPS環境、またはGitHub Pagesでの公開が推奨です。
